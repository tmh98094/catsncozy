# Data Folder

This folder contains all your website data in JSON format.

## Files:
- `cats.json` - All cats available for adoption
- `testimonials.json` - Customer testimonials
- `services.json` - Boarding service packages

## How it works:
1. Your admin panel reads from these files
2. When you make changes, they're saved back to these files
3. GitHub keeps version history of all changes
4. Works across all devices

## Important:
- Don't delete these files
- Keep the JSON format valid
- GitHub automatically backs up all changes
